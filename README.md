# KitchenBlue 🍳
- A Webapp for Recipe Search. 
- It searches for recipes by ingredients and dish name contents.

# Technical Info 🔧
- Languages used:- HTML5, CSS3, JavaScript.
- Restful Api, Postman, 🐶RecipePuppy Api
